Mac终端类

安装ADB——fastboot刷机走起

使用brew

终端里一条命令：

brew install Caskroom/cask/android-platform-tools

测试是否安装成功：

adb

如果出现命令提示，而不是“⌘ not found”，即安装成功。

同时安装Python2.7和Python3.7

使用brew：

brew install python3

因为Mac自带Python2.7，所以使用brew安装Python3即可完成共存。
千万不要卸载Mac的Python2.7，会造成系统严重Bugs，因为Mac系统很多程序依赖自带的Python2.如LXcode

终端通过代理联网

export ALL_PROXY=socks5://127.0.0.1:1086

127.0.0.1是本地代理，1086是代理端口

隐藏文件显示/隐藏
